## Proyectos Destacados

### Clasificador de Residuos con IA
Sistema de clasificación inteligente de residuos mediante Deep Learning. Procesamiento de 50,000 imágenes para entrenar un modelo CNN con TensorFlow/Keras, alcanzando 94% de precisión en la identificación de 5 categorías de residuos, con inferencia en tiempo real por debajo de los 100ms.

`Python` `TensorFlow` `Keras` `OpenCV` `Machine Learning`

### Detector de Acné Médico
Herramienta de análisis dermatológico basada en procesamiento de imágenes médicas. Aplica visión por computadora para detectar y clasificar tipos de acné, con generación automática de reportes en PDF y gestión de historial de pacientes.

`Python` `PyQt` `OpenCV` `Image Processing`

### App Móvil CIE-10 MINSA
Aplicación móvil para consulta del catálogo CIE-10 usada por profesionales de la salud. Transformación de datos CSV a una base SQLite optimizada, con motor de búsqueda por debajo de 500ms y funcionamiento completamente offline.

`Android` `Java` `SQLite` `Data Processing`

### Sistema Web - Área de Terapia (MINSA Talara II)
Sistema web para la gestión de pacientes del área de terapia, con almacenamiento y análisis de datos, reportes estadísticos y control de acceso por roles de usuario.

`PHP` `MySQL` `HTML/CSS` `Reporting`

### Sistema CRUD de Estudiantes
Sistema web con operaciones CRUD completas para la gestión de registros de estudiantes, validación de datos y almacenamiento estructurado en MySQL.

`PHP` `MySQL` `Data Validation`

### App Móvil para Cálculo de IMC
Aplicación Android para el cálculo y seguimiento del Índice de Masa Corporal, con historial de datos, análisis de tendencias y recomendaciones basadas en datos.

`Android` `Kotlin` `SQLite` `Analytics`
